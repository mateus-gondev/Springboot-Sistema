## Comandos Úteis

Realizar o build do projeto.
~~~ java
mvn clean package -D skiptests
~~~

Executar o projeto localmente.
~~~ java
java -jar target/Springboot-Sistema-Acadêmico-0-1.0-SNAPSHOT.jar
~~~
